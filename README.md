# Misi Terakhir

Mini game cerita singkat berbasis HTML, CSS, dan JavaScript murni. Dibuat sebagai ucapan terima kasih hangat dari 3EA05 untuk KP Clara dan Tutor Dhiyaa.

## Menjalankan secara lokal

Buka [index.html](index.html) langsung di browser.

## GitHub Pages

Proyek ini hanya memakai file statis, jadi bisa dipublikasikan dengan mengunggah folder ini ke GitHub lalu mengaktifkan GitHub Pages dari root repository.
